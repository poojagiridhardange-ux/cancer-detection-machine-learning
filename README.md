# Cancer Detection using Machine Learning

## Project Overview

This project explores how data analytics and machine learning techniques can help improve cancer detection. The study focuses on identifying patterns in healthcare data that may assist in predicting whether a patient has cancer.

The research applies predictive analytics and machine learning models to analyze patient-related data and classify cases as **cancer** or **no cancer**. The aim is to demonstrate how data-driven approaches can support healthcare professionals in improving diagnostic accuracy and patient outcomes. 

This project was completed as part of the **Bachelor of Business Administration (Business Analytics)** program.

---

## Research Title

**The Analytics Alchemy: Transforming Data into Cancer Detection Insights** 

---

## Problem Statement

Traditional cancer diagnosis techniques rely on methods such as imaging, biopsies, and histopathological analysis. These methods provide valuable insights but may sometimes be time-consuming and subject to human interpretation.

This study explores the use of **machine learning and predictive analytics** to identify patterns in medical data and assist healthcare professionals in detecting cancer more effectively. 

---

## Research Objectives

The study aims to:

* Identify medical data categories that contribute to early cancer detection
* Evaluate the role of machine learning and artificial intelligence in improving diagnostic accuracy
* Analyze patterns and trends in healthcare datasets related to cancer
* Examine how predictive analytics improves the speed and accuracy of diagnosis 

---

## Dataset

The research uses a **synthetic dataset containing 10,000 patient records** designed to simulate real medical data.

Important variables included in the dataset:

* Age
* Gender
* Family history of cancer
* Smoking habits
* Alcohol consumption
* Diet quality
* Physical activity
* Radiation exposure
* Chronic illness
* Laboratory results
* Imaging results
* Cancer diagnosis (target variable)

These variables help analyze demographic, lifestyle, and medical factors associated with cancer detection. 

---

## Methodology

The research follows a **quantitative research design** and uses multiple machine learning models to classify cancer outcomes.

Main steps in the analysis:

1. Data preprocessing
2. Descriptive statistical analysis
3. Feature selection
4. Machine learning model training
5. Model evaluation

Machine learning algorithms considered in the study include:

* Logistic Regression
* Decision Tree
* Random Forest
* Support Vector Machine (SVM) 

The dataset was divided into **training and testing sets using an 80/20 split** for model evaluation.

---

## Data Analysis Techniques

Several analytical techniques were used in this study.

### Descriptive Statistics

Descriptive statistics were used to summarize dataset characteristics using:

* Mean
* Median
* Mode
* Standard deviation
* Variance

These measures helped understand patterns in variables such as age distribution, lifestyle behavior, and health indicators related to cancer risk. 

### Feature Selection

Statistical tests were used to identify the most significant predictors influencing cancer detection.

### Dimensionality Reduction

To simplify complex data while preserving patterns, the following methods were used:

* Principal Component Analysis (PCA)
* t-Distributed Stochastic Neighbor Embedding (t-SNE) 

---

## Decision Tree Model

A **decision tree classification model** was used to predict cancer detection.

The model divides the dataset into branches based on different medical and lifestyle factors. Each branch represents a decision rule that eventually leads to a final classification:

* Cancer
* No Cancer

Decision trees are widely used in healthcare analytics because they are easy to interpret and visualize. 

---

## Model Performance

The predictive model achieved strong performance results.

Key metrics reported in the study:

* Accuracy: **93%**
* Precision: **90.3%**
* Recall: **99.6%**
* F1 Score: **93.5%**
* AUC-ROC Score: **0.91**

These results indicate that the model can effectively distinguish between cancer and non-cancer cases. 

---

## Key Findings

The analysis identified several factors influencing cancer risk, including:

* Age
* Family history of cancer
* Smoking habits
* Alcohol consumption
* Body Mass Index (BMI)
* Radiation exposure

Certain biomarkers such as **CA-125 and PSA** were also identified as important predictors in cancer detection models. 

---

## Limitations

The study acknowledges some limitations:

* The dataset used was synthetic rather than real patient data
* Real-world clinical validation of the model was not conducted
* Synthetic data may not fully represent real patient diversity

Future research could improve the model using real healthcare datasets and additional medical variables.
